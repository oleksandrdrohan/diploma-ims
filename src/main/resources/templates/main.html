About IMS

This site is my diploma project and my first "big" project. This is a prototype of a very simplified warehouse and order management system. It is designed for a small warehouse for 4-6 people. Which I have worked on in the past.

To write this app I used:

  • Java
  • Spring IoC
  • Spring MVC
  • Spring Security
  • Spring Data JPA
  • Spring Boot
  • Thymeleaf
  • MySQL
  • a little bit of JS

The basic mechanics of a web application are:

  1. CRUD operations with items.
  2. Pagination.
  3. Ability to create an order for stock replenishment or shipment.
  4. Authorization through Spring Security.
  5. Search of products and orders.
  6. Viewing the update of the number of items.
  7. Automatic replenishment/shipment of items specified in the order (change status).

The logic of this application is as follows. The manager or head of the warehouse adds goods that appear in the warehouse or will arrive in the future. Then they create an order, if it is a request for replenishment - then when the machine with the goods arrives, the workers will check it and change its status, which will automatically update the quantity of the goods. The same with the shipment.

IMPORTANT!!!

To use the ordering functionality, you need to create an item to begin with! According to my logic, you need to add items first, and then replenish them. That is, if you know that some product will arrive, first add it and set the quantity = 0.
